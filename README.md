Osmo Supply Clone
A clone of Osmo Supply using Angular, Next.js, and JavaScript, with a Video Call Feature powered by LiveKit. The frontend is built with Angular for dashboards and Next.js for SSR. Install dependencies with npm install, then run ng serve for Angular and npm run dev for Next.js. The video call feature enables high-quality real-time communication using LiveKit.
